My Projects

Predicting House Pricing with Linear Regression and Adaboost (Python)

For my Machine Learning final project, my group implemented Linear Regression and Adaboost learning models to predict propety pricing in the Boston area. The goal of this project was to a) comapre the accuracies of the two models, and b) evaluate which features regarding the properties weighed more in the models' prediction. We found that (not surprisingly) the AdaBoost model was much more accurate than Linear Regression (~0.9 and ~0.7, respectively). As for the factors most valued by the models, both weighed the average number of rooms highly, but differed drastically in their use of other features (or lack thereof).

Big-Int Data Type (C++)

The Big-Int data type that I created was designed to store large values in C-style arrays, where each index held 3 digit values at different memory addresses. It was very difficult to be able to delete and allocate memory at the appropriate times, especially when implementing the copy constructor and = operator. This project drastically improved my knowledge and understanding various concepts regarding memory.

WORDLE Recreation (Java)

For my Data Structures final project, I decided to create a variation of WORDLE. I first imported 3 CSV files with 3, 4, and 5 letter words to be randomly selected from and guessed. I used a comparison algorithm to provide feedback at each of the users' guesses with correct and incorrect letters/ placement of letters. The most difficult part was dealing with words with repeated letters. I then implemented a Binary Search Tree to implement a leaderboard.

What to Eat (Python)

In my first personal project, I created a program that would help me with something I struggle with every day: finding something to eat. It takes in a user-inputted list of available ingredients and returns a list of potential meals that can be made with them. It uses a Food class with attributes that will help in the sorting/comparison algorithm: meal time, meal name, and ingredients attributes.
